# 16 · 全国统一市场框架 Unified National Market Framework

## 布局方式

**3×2 成就卡片网格** 为主体，顶部标题区：

- 顶部：页面标题 + 引言说明
- 主体：6 个建设成就卡片，每卡含图标、数字、说明文字

## 重点内容

| 卡片 | 成就 |
|---|---|
| ① 规则体系 | "1+6"市场基本规则发布，制度框架完整确立 |
| ② 跨区机制 | 跨经营区交易机制批复（国发〔2025〕915号） |
| ③ 省间互济 | 4,000万千瓦省间互济峰值记录 |
| ④ 两个全覆盖 | 中长期31省全覆盖 / 现货5+18试运行 |
| ⑤ 绿色市场 | 绿色电力2,174亿kWh / 新能源入市1.5万家 |
| ⑥ 新型主体 | 储能/虚拟电厂等1.5万家新型主体参与市场 |

## 动效规划

| 类 | 元素 | 延迟 |
|---|---|---|
| `.au .d1` | 标题 | 0.1s |
| `.au .d2` | 引言文字 | 0.2s |
| `.ai .d3`～`.d8` | 六个成就卡逐一淡入 | 0.3s–0.8s |
