# 05 · 装机容量结构 Installed Capacity

## 布局方式

**左侧文字卡片 + 右侧 SVG 甜甜圈图** 两栏布局（`grid 1.2fr 0.9fr`）：

- 左栏：大要点标题 + 按能源类型排列横向条形图（太阳能/煤电/风电/水电/核电/气电）
- 右栏：SVG donut 图展示清洁 vs 非清洁占比 + 利用小时数对照表

## 重点内容

| 指标 | 2025年数值 |
|---|---|
| 全国总装机 | 30.4亿千瓦（约304,000万千瓦） |
| 清洁能源装机占比 | **59.4%**（约180,703万千瓦） |
| 太阳能装机 | 9,480万千瓦（增速最快） |
| 煤电装机 | 约12,400万千瓦 |
| 风电装机 | 约5,100万千瓦 |
| 水电装机 | 约4,200万千瓦 |
| 核电装机 | 约643万千瓦 |

利用小时数（2025）：

| 类别 | 利用小时数 |
|---|---|
| 核电 | 7,664 |
| 水电 | 3,567 |
| 火电 | 4,093 |
| 风电 | 2,323 |
| 太阳能 | 1,298 |

## 动效规划

| 类 | 元素 | 延迟 |
|---|---|---|
| `.au .d1` | 页面标题 | 0.1s |
| `.au .d2` | 清洁能源大数字横幅 + 条形图 | 0.2s |
| `.au .d3` | SVG donut 图 | 0.3s |
| `.au .d4` | 利用小时数表格 | 0.4s |

SVG donut 使用 `stroke-dasharray` 绘制，中心显示59.4%大字。
